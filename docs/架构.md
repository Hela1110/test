# 客户端-服务器购物系统项目架构

## 项目概述
本项目是一个基于Qt的客户端-服务器购物系统，实现了用户注册登录、商品浏览购买、购物车管理、订单历史、客服聊天等功能。

## 目录结构
```
project/
├── client/                    # 客户端代码
│   ├── include/              # 头文件
│   │   ├── login/           # 登录相关
│   │   ├── mainwindow/      # 主窗口相关
│   │   ├── shopping/        # 购物相关
│   │   └── chat/            # 聊天相关
│   ├── src/                 # 源代码
│   ├── resources/           # 资源文件
│   │   ├── images/         # 图片资源
│   │   ├── qss/            # 样式表
│   │   └── icons/          # 图标资源
│   └── ui/                  # UI文件
│
├── server/                   # 服务器代码
│   ├── include/             # 头文件
│   │   ├── database/       # 数据库操作
│   │   ├── network/        # 网络通信
│   │   ├── manager/        # 业务管理
│   │   └── thread/         # 线程池
│   └── src/                # 源代码
│
└── database/                # 数据库设计文件
    ├── schema/             # 数据库表结构
    └── sql/                # SQL脚本
```

## 模块说明

### 1. 客户端模块

#### 1.1 登录注册模块
- 用户登录界面
- 新用户注册
- 密码验证
- Socket连接管理

#### 1.2 主界面模块
- 轮播图展示
- 商品推荐
- 促销活动展示
- 商品搜索
- 风格切换

#### 1.3 购物功能模块
- 商品详情展示
- 购物车管理
- 订单生成
- 支付处理

#### 1.4 用户中心模块
- 个人信息管理
- 订单历史记录
- 客服聊天
- 设置功能

### 2. 服务器模块

#### 2.1 商品管理系统
- 商品信息管理
- 库存管理
- 促销活动管理
- 商品搜索服务

#### 2.2 用户管理系统
- 用户账户管理
- 权限控制
- 用户信息维护

#### 2.3 订单处理系统
- 订单生成
- 订单状态管理
- 支付处理
- 退货处理

#### 2.4 数据统计系统
- 销售数据统计
- 用户行为分析
- 报表生成

#### 2.5 网络通信模块
- 多线程处理
- Socket通信
- 数据传输加密
- 线程池管理

### 3. 数据库设计

#### 3.1 数据表设计
- Client表：用户信息
- Product表：商品信息
- Order表：订单信息
- Product_type表：商品类型

#### 3.2 表关系
- Product与Product_type：一对多关系
- Client与Order：一对多关系
- Product与Order：一对多关系

## 技术特点
1. 采用Qt框架开发，实现跨平台支持
2. 使用异步Socket通信
3. 实现多线程处理机制
4. 采用线程池优化并发性能
5. 使用QSS实现界面美化
6. 支持实时聊天功能
7. 实现数据可视化展示
